O projeto Healing é um site que simula cadastro de usuários (pacientes e medicos) com todas as funcionalidades relacionadas a consultas e separados por área do paciente e do médico

Área do Paciente
- Cadastro
- Buscar de médicos por nome e/ou especialidade
- Agendamento de consulta
- Consultar as consultas agendadas e status
- Constulta de documentos e opção de download
- Efetivação da consulta online
- Cancelamento de consulta


Área do Médico
- Cadastro
- Abrir horários disponiveis para consulta
- Consultar as consultas agendadas e status
- Constulta e inclus~]ao de documentos
- Efetivação da consulta online
- Cancelamento de consulta

Esse projeto foi desenvolvido durante a jornada Pystackweek10 do canal Pythonando
Foi um intensivo de python com Django abordando os contaúdos:

- Metodos
- Classes
- Auth admin do Django
- Banco de dados (SQLite)
- URLs
